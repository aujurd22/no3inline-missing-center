# Direction ④ — 全局共线能量下界（形式化）

**问题**：是否存在 C4-对称的 m=37 无三点共线配置？

## 核心思路

定义函数 E(G) ≥ 0，G 为 m-vertex 2-因子，满足 E(G) = 0 ⇔ G 是 rot4-NTIL 解。
证明对所有 37-vertex 2-因子有 E(G) ≥ c > 0 → 不存在解。

## 候选能量函数

### E1: 线占用二阶矩

设每条几何线 ℓ 占用 o_ℓ 个 C4 提升点（ℓ 由 2m×2m 网格上两点确定）。
定义 E₁ = Σ_ℓ max(0, o_ℓ - 2)。

- 对 m=36 解：E₁ = 0（全部 o_ℓ ≤ 2）
- 对随机 m=37 候选：期望 E₁ ≈ 10.3m（来自 R9e 分析）

**问题**：E₁ 是全局的、难以下界。需转向代数/组合结构。

### E2: 二次型零点计数（R8 框架）

在 R8 等价框架中，NTIL ⇔ m 个 2-因子边满足 O(m³) 个二次等式 det(i,j,k) ≠ 0。
每一组 det ≠ 0 对应二次型的非零点。

定义 E₂ = #{triples (i,j,k) : det(i,j,k) = 0}。则 E₂ = 0 为解。

**方法**：对每个 2-因子 G，det(i,j,k) 是顶点坐标的函数，具有：
- 三元一次/二次型结构（行列式展开后为线性/双线性）
- 对 C4 旋转有不变量性质

**拦路虎**：O(m³) 个约束在代数几何上是 0 维理想（已由完全确定性证明），局部剪枝在 m=17 后无意义（T14）。

### E3: 斜率多重集的组合下界

每条边 (i,j) 对应奇数坐标 (a,b) = (2(m-i)-1, 2(m-j)-1)。
斜率 s = (a_i - a_j)/(b_i - b_j)（来自 C4 提升的 4 个像点方向）。

4m 点确定 ~C(4m, 2) 条方向。每个方向最多承载 ceil(C(4m, 2)/唯一方向数) 条点对。
若有方向承载 ≥3 个点对，则可能有共线。

**关键观察**：C4 对称将方向分成 4-轨（或 8-轨），方向数量远少于 C(4m, 2)。

**上界**：2m×2m 网格中唯一斜率数 = O(m²)。
对 m=37：~1369 CELLS → 约 10K C4 提升点对 → 约数百个 C4-轨道方向。

若一个方向有太多点对，则必有 o_ℓ ≥ 3。

### E4: 偶数-奇数层相互作用（新）

C4 对称在奇数坐标下：
- 奇数坐标 (a,b) 的奇偶性：a,b 均为奇数 ⇒ (a,b) 在 "奇数层"
- C4 旋转保持奇偶性：(-b, a) — b 为奇数 ⇒ -b 为奇数 ✓
- 2m×2m 网格有奇偶两类点

**猜想**：C4 对称强制奇数层点与偶数层点在每条线上交替出现，从而对每条线 o_ℓ 的奇偶性有约束。这个约束在 m 从偶→奇时发生变化，可能产生 m=37 独有的奇偶障碍。

## 推荐优先级

1. **先做 E3 的数值实验**：对 m=10…30 的已知解，统计方向/斜率多重集的最大占用，看是否随 m 单调逼近 3。若在 m=36 时最大占用 = 2（安全），对 m=37 外推 → 可判断 m=37 是否"必然超过 2"。
2. **E4 的奇偶论证**：形式化"奇偶层交替"定理，这个是纯数学工作。

## 结论

方向④是目前唯一未穷尽的数学方向，且有可能一击定音（证明 m=37 不可解）。但它是**纯数学研究**，不依赖计算，需要新的定理。本会话阶段的工作到此收口。

**五个方向总评**：
- 方向①③⑤：已穷尽并排除（各给出致命反例/不收敛证明）  
- 方向②（有界手术）：关键发现——m→m+1 的 C4 几何发生根本变化（N 变化改变所有点的位置），种子解不可携带，方向②不成立
- 方向④（能量下界）：最后的数学突破机会，需独立形式化研究

# Direction ④：全局共线能量下界（完整形式化）

## 概述

目标：构造能量函数 E(G) ≥ 0，G 为 m-顶点 2-正则有向图，满足
E(G) = 0 ⇔ G 是 rot4-NTIL 解。
然后证明对所有 m=37 的 2-因子有 E(G) > 0。

## 基础几何设定

- 网格大小 N = 2m，坐标 (x,y) ∈ [0,N-1]²
- 基本象限 Q = {(x,y): 0 ≤ x < m, 0 ≤ y < m}
- C4 旋转: (x,y) → (N-1-y, x) → (N-1-x, N-1-y) → (y, N-1-x)
- 第 i 个 cell 的坐标记为 (x_i, y_i) ∈ Q
- 提升点集: P = {C4^r(x_i, y_i): i=1..m, r=0..3}, |P| = 4m
- 方向键 (A,B,C): line_of(p,q) = 归一化的 Ax+By+C=0

## E1: 线占用矩（已测，非瓶颈）

略。E[B] ≈ 10.3m，但未证明无法为零。

## E2: 二次型零点（R8 框架，非瓶颈）

O(m³) 个 det(i,j,k) ≠ 0 约束构成 0 维理想。m≥17 后计算不可行。

## E3: 方向键唯一性与鸽笼原理

### 定理 E3.1（方向键单射性）
在有效 NTIL 解中，每个方向键 (A,B,C) 恰好对应唯一的点对 {p,q} ⊆ P。
证明：若两对 {p,q} 和 {r,s} 给出相同 (A,B,C)，则四点共线 → 违反 NTIL。

### 系 E3.2
有效 NTIL 解需要至少 C(4m,2) 个互异的方向键。

### 实验 E3.3（已知解验证）
对 m=5..19 及 m=36 的所有已知解，`distinct_keys = C(4m,2)`，ratio = 1.0000。
证实 NTIL ⇔ 方向键单射。

### 实验 E3.4（方向键总空间）
对 N×N 网格枚举所有互异方向键：
| m | N | 总方向键数 | 所需 C(4m,2) | ratio |
|---|---|---|---|---|
| 5 | 10 | 2,306 | 190 | 0.082 |
| 10 | 20 | 36,510 | 780 | 0.021 |
| 14 | 28 | 139,966 | 1,540 | 0.003 |

### 结论 E3.5
方向键总空间 O(m³) 远超所需 O(m²)。ratio → 0 随 m 增长。
**鸽笼原理不适用**。

## E4: C4-轨道分析

### 定理 E4.1（C4-轨道方向键）
方向键 (A,B,C) 在 C4 旋转下变换为:
(A,B,C) → (-B, A, C - B(N-1))
（后续需符号归一化。）
大部 Key 属于 4-轨道，少数属于 2-轨道（C4-不变线）。

### 实验 E4.2（C4-轨道计数）
| m | N | C4-轨道数 | 最少所需(÷4) | ratio |
|---|---|---|---|---|
| 5 | 10 | 2,020 | 48 | 0.024 |
| 10 | 20 | 31,890 | 195 | 0.006 |
| 14 | 28 | 122,394 | 385 | 0.003 |

轨道分布压倒性的 4-轨道，2-轨道 ≤ 23 个。

### 结论 E4.3
C4-轨道空间也远超所需。**非瓶颈**。

## E5: 模约束与奇偶分析（新）

### E5.1: 奇偶坐标下的 C4 作用

将 cell (x,y) 映射为奇数坐标 (a,b) = (2(m-x)-1, 2(m-y)-1)。
此时 a,b ∈ [-2m+1, 2m-1] 且均为奇数。

C4 旋转在奇数坐标下的作用：
(a,b) → (2m-1-b, a) → (2m-1-a, 2m-1-b) → (b, 2m-1-a)

奇偶性变化：(odd,odd) → (even,odd) → (even,even) → (odd,even) → (odd,odd)
每个 cell 在每个奇偶类恰好贡献 1 个点。

### 定理 E5.1（对角约束）
在 NTIL 解中，全网格的两条对角线 (y=x, y=N-1-x) 上各至多 2 个 C4-提升点。
因此**基本象限 Q 中至多 1 个 cell 位于对角线 x=y 上**。
证明见方向②对角分析。已在 m=5..19 解中验证（k ≤ 1）。

### E5.2: 坐标模约束

设对素数 p，考虑 cell 坐标 (x_i, y_i) mod p。
若 p ≤ m，则 (x_i mod p, y_i mod p) 在 F_p² 中。

C4 旋转在 mod p 下的作用（N=2m）：
x' ≡ -1-y (mod p) (因 N≡0, N-1≡-1 mod p)
y' ≡ x (mod p)

这是 F_p² 上的 4-循环 (x,y) → (-1-y, x) → (-1-x, -1-y) → (y, -1-x) → (x,y)。

**观察**：若 m 被 p 整除，则 N=2m≡0 mod p，N-1≡-1 mod p。
若 m 不被 p 整除，则 N mod p ≠ 0，变换更复杂。

### 猜想 E5.3（未证明）
对 m=37（素数），F_37 上的 C4 旋转结构可能产生额外的行列式约束。
m=36=2²·3² 的因子结构与 m=37（素数）不同，可能对应不同的行列式零分布。

### E5.4: 和/差约束

对每条边 (i,j)（cell i 连到 cell j），其奇数坐标表示：
a_i = 2(m-x_i)-1, b_i = 2(m-y_i)-1
a_j = 2(m-x_j)-1, b_j = 2(m-y_j)-1

差 a_i - a_j = -2(x_i - x_j)，因此是 2 的倍数。
**所有奇数坐标差均为偶数**。

这一简单事实在共线性判定中的作用：
det(C4^p(i), C4^q(j), C4^r(k)) 是四个 2×2 行列式之和，每个均为偶数。
因此 det 本身是偶数。当 det=0 时，是偶数零——无非平凡模 2 约束。

### 猜想 E5.5（奇偶层交替，未证明）
C4 奇数坐标的奇偶性在每条线上交替：
若线 ℓ 上有奇数类点 p 和 q，则 ℓ 上也必然有偶数类点 r，
或 ℓ 的斜率/截距具有奇偶特征能使各类点共存。
该猜想在已知解中成立（所有 0 坏线），对 m=37 是否必然违反未知。

## E6: 行列式二次型分析（理论方向）

### 定义
对三元组 (i,j,k) 和旋转组合 (r,s,t) ∈ {0,1,2,3}³，
det(r,s,t,i,j,k) = 行列式[ C4^r(x_i,y_i), C4^s(x_j,y_j), C4^t(x_k,y_k) ]

### 定理（R8）
NTIL ⇔ 对所有 collinear triple (i,j,k) 和所有旋转组合 (r,s,t)，
det(r,s,t,i,j,k) ≠ 0。

### 观察
det 是 cell 坐标 (x_i,y_i) 的二次多项式，具有对称性。
对固定 (r,s,t)，det = 0 是 Q 中 cell 坐标的二次方程。
全部 O(m³) 个 det ≠ 0 条件构成代数系统。

### 猜想 E6.1（二次型零点约束，未证明）
对所有已知 NTIL 解，每个 det=0 的二次方程在 Q 中有唯一解（对应被禁止的三元组）。
若 m=37 时这些二次方程的零点集覆盖所有 2-因子，则解不存在。
这是将问题翻译为代数几何中的"理想成员判定"问题。

## 综合结论

### 已排除的瓶颈
1. ❌ 方向键总数（E3）：ratio → 0
2. ❌ C4-轨道数（E4）：ratio → 0
3. ❌ 方向键单射性：等价于 NTIL，非新约束
4. ❌ 奇偶模 2 分析（E5.4）：行列式恒为偶数，无非平凡模 2 约束

### 未排除但未证明的方向
1. ⚠️ 坐标模 p 分析（E5.2）：m=37 为素数时的 F_37 结构不同
2. ⚠️ 二次型零点覆盖（E6）：是否所有 2-因子必经某个 det=0
3. ⚠️ 奇偶层交替（E5.5）：未形式化

### 最终评估
方向④（能量下界）在本次系统分析中**未能产生可证明 m=37 不可能的定理**。
所有鸽笼/计数类的论证都不适用（容量过剩）。
代数几何路线（二次型零点）可能仍有希望，但需要完全的 Gröbner 基或消元理论，
这超出了本会话框架的范围。

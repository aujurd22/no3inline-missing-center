## 1. 方向 1（局部引理 / Switching 存在性证明）—— 值得测，但框架要改

### 1.1 关键修正：vanilla LLL 在「单一随机 2-因子」上必然失效

LLL 要求坏事件之间只有**有界依赖度 d**（依赖图邻接 ≙ 不独立）。

若概率空间是「抽一个均匀随机 2-因子 G + 随机取向」，则**所有坏事件都是同一个随机对象 G 的函数** ⇒ 任意两个不同坏事件都不独立 ⇒ 依赖度 d = N−1（N=坏事件总数）。此时 LLL 条件退化为 `e·p·N ≤ 1`，即要求 `p ≤ 1/(e·N)`。

而 R9e 已给出 `E[#bad] = Σ P(A_i) = N·p̄ ≈ 10.3m`。若 N ~ O(m³)（4m 点可确定的全部几何线），则 `p̄ ≈ 10.3m / m³ = O(1/m²)`，于是
`e·p̄·N ≈ e·(10/m²)·m³ = 10e·m ≫ 1/e`（m=37 时约 1000× 超出）。

**结论：vanilla LLL（在「一个随机 2-因子」空间上）不可用**，不是差几个数量级，而是量级错误。

### 1.2 真正可行的 LLL 载体（评审已点到，需明确）

- **熵压缩 / Moser–Tardos 算法（局部重采样）**：把概率空间改成**乘积空间**——每个「可独立重采样」的局部部件（如一条 alternating-cycle switch）是一组独立随机变量。对每个坏事件，只重采样它依赖的少数 switch。这样依赖度 d 由「共享 switch 的坏事件数」决定，可做到有界。
- **Switching argument（配置模型）**：直接证明「若某个 2-因子有坏三元组，则存在一个局部 k-edge switch 严格减少坏的计数」⇒ 由极小反例导出矛盾。这是单随机对象下最自然的「存在性」证明工具。
- **Lopsided / cluster-expansion LLL**：当普通 LLL 常数不够时放松独立性假设。

### 1.3 诚实风险（我的量级估计）

即便改用熵压缩 / switching，真正的瓶颈是**依赖度 d 仍可能很大**：一条几何线 L 经过 3 个提升点，每个点又在 O(m) 条其他线上 ⇒ 与 L 共享点的坏事件数 d ~ O(m)。要让 LLL/switching 成功，需要 `p·d ≲ 1`，即 `p̄ ≲ 1/m`。而 first-moment 给的 `p̄ ~ O(1/m²)` 是**平均**值，最坏线（高频斜率）的 p 可能大 1–2 个量级 ⇒  borderline，可能差 ~10×–100×。这正是评审「止损标准」要测的东西。

### 1.4 裁决

**做 gating 实验**（廉价，1–2h）：在 m=10–30 精确测
1. 每个几何坏事件的 `p`（在随机 2-因子 + 随机取向模型下）；
2. 依赖度 `d`（共享 ≥1 提升点的坏事件数，及共享 switch 的坏事件数）；
3. `e·p·(d+1)` 与 `e·p̄·(d+1)`；
4. **switch-邻域 reducibility**：随机取有坏的配置，枚举其 k-edge switch（k=2,3,4），统计「能严格减少 bad 计数」的比例。

若 `e·p·(d+1)` 经 lopsided 仍 ≫1 且 switch 不可降，则**暂缓**方向 1，转方向 3。

## 2. 方向 2（m=36 有界 2-因子手术）—— 新且构造性，中程主力

- 数据齐备：m=3..36 全部有解（`results/solutions/`）。
- **关键提醒（与记忆一致）**：简单「+1 cell」包含扩张**不存在**（已有记录），必须做完整 k-edge switch / ejection chain / alternating-cycle replacement。
- 价值双轨：找到解 ⇒ m=37 解决；证明「所有 m=36 解在有限手术半径内均不可扩张」⇒ 新结构定理（非存在性的局部证据）。
- 风险：k-edge switch 空间随 k 增长快；每个候选需增量维护 126 万条 per-line 二次约束。可与方向 3 的 lazy 模型联用（lazy 给出候选 2-因子，手术在其上做）。
- **裁决**：中程必做，但排在 gating + lazy 原型之后。

## 3. 方向 3（Lazy constraint generation / branch-and-cut）—— 计算路线头号优先级

- 直击我们已诊断的死因：CP-SAT 在 m≥17 撞 **presolve / 全量 126 万约束墙**（时段13）。
- 我们已有：R9b 快速生成器（reps→points→directions，O(m²·|G|·|D|)）+ `verify_cells` 真值。只需**重构求解循环**：
  1. 初始模型只放 2-因子 + Sidon + 基帧 3-free + 少量高频障碍；
  2. 求候选 2-因子 ⇒ 提升 ⇒ 找实际违线；
  3. 只把 violated-line 约束成批加入；
  4. 热启动重复 ⇒ branch-and-cut / constraint-generation 循环。
- 附带红利：自动揭示「真正活跃」的障碍族（直接喂给方向 5 的定理猜测）。
- **止损标准（评审给得好）**：m=17–22 比较全量 vs lazy；若迭代后仍累积到几十万约束且无时间优势 ⇒ 停。
- **裁决**：**最高 ROI 的计算路线**，原型成本极低（复用现有代码），优先做。

## 4. 方向 4（全局「共线能量」下界）—— 新理论角度，长线彩票

- 我们目前所有定理都是**局部**必要条件（Sidon / per-line ≤2 / 基帧 3-free / 环型）。缺全局不变量。
- 干净的重述：定义 `E = Σ_L C(t_L, 3)`（对所有几何线、t_L=该线占用点数）。rot4-NTIL 存在 ⇔ ∃ 2-因子使 E=0。若能证明「任何 37 顶点 2-因子在 C4 提升下总有 E≥1」⇒ m=37 非存在。
- 工具：离散傅里叶 / 加性组合 / 整数点关联界 / 多项式法。同意**不做完整 Gröbner**（变量/不等式规模太大）。
- **裁决**：最有「一击定音」潜力的非存在性路线，但最难、最不确定。作为低优先级并行长线，先做一个小第一步：形式化定义能量并尝试对 m=37 给出下界（哪怕只是数值/启发式下界）。

## 7. 推荐执行序列（ROI 排序）

1. **Gating 实验（方向 1 止损）** — 测 p、d、ep(d+1)、switch-reducibility（m=10–30）。廉价、1–2h，一锤定音方向 1 是否值得继续。
2. **Lazy CP-SAT 原型（方向 3）** — 在 m=17–22 跑全量 vs lazy 对照。复用现有生成器/校验，成本极低。
3. **数据驱动不变量挖掘（方向 5）** — 与 (1) 共享代码，并行。
4. **据 (1)(2) 分叉**：
   - 若 LLL/switching 可行 ⇒ 攻方向 1 存在性定理（目标：对充分大 m 的存在定理 + m=37 有限边界）；
   - 若 lazy 有优势 ⇒ 推到 m=37 真实攻击；
   - 方向 2（m=36 手术）作为构造性后续；方向 4（能量下界）作为长线彩票。

**一句话**：计算主攻 lazy（方向 3），理论主攻 switching/熵压缩 LLL（方向 1，先 gating），两者用方向 5 的数据互相校准；方向 2/4 为后续梯队。
